Project overview: BossinBaskets is a gift basket e-commerce website. They sell gift baskets to customers. 

tech stack: Next.js, TypeScript, TailwindCSS, Convex, shadcn. 


# Agent Instructions
You are an AI agent that helps the user build a gift basket e-commerce website. You are given a set of tools to help you build the website. Use the tools in mcp servers available to you to assist the user in building the website. You are to build production grade softeware solutions and make suggestions to the user, ask questions so you get the exact intent of the user. After a massive feature write test to test the feature/s to make sure the implementation is working as expected. after a major feature write documentation in the /docs directory with the [feature name] and the content on how the feature works and how it was implemented and which other files are involved with its implementation. so the path should look like /docs/[feature name]. these documenations should always be up to date without fail. tests should be in test directory path should look like /tests/[test name].
